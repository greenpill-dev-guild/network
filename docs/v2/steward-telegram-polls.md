# Greenpill Steward Telegram Questions

## Intro message
Stewards, after a closer read of the workshop notes and forum thread, we already seem aligned on a few things: newcomer-first framing, keeping the map or explorer, avoiding a full redesign, making onboarding more human, and not forcing Web3 complexity up front. So instead of polling every UI choice, I want to focus on the few decisions that still feel genuinely open.

Please vote on the 3 polls below, then reply to the final nomination prompt.

## Poll 1
**Question:** What should a non-Web3 visitor understand in the first 30 seconds?

**Options:**
- Greenpill is a living network of local chapters, builders, and storytellers
- Greenpill is a home for builders creating tools for regenerative coordination
- Greenpill is a learning gateway into public goods and regenerative action
- Greenpill is a research and sensemaking hub for regenerative coordination and public goods

**Recommended direction:** Greenpill is a living network of local chapters, builders, and storytellers

## Poll 2
**Question:** What should do the most work to prove Greenpill is active right now? Pick up to 3.

**Options:**
- Featured chapter stories and local examples
- Builder tools, guild output, and active projects
- Current programs and community calls
- Impact signals like outcomes, stats, and recent wins
- Human stories from stewards, members, and partners
- The map or explorer as a living view of the network
- Books, podcast, and learning content as the visible knowledge layer

**Recommended direction:** Featured chapter stories and local examples + Builder tools, guild output, and active projects + Current programs and community calls

## Poll 3
**Question:** How should Greenpill Garden appear in the first release?

**Options:**
- Treat it as a flagship current program that shows where the network is headed
- Treat it as one important program inside a broader network story
- Treat it as the clearest example of steward support and development
- Treat it as a bridge between chapters, tools, and the wider regen ecosystem
- Keep it lighter for now until the program direction is more settled

**Recommended direction:** Treat it as one important program inside a broader network story
