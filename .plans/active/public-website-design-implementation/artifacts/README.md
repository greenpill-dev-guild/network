# Public Website Design Implementation Artifacts

This folder contains temporary design, strategy, workshop, and prompt material that supports the public website implementation.

- `hifi/`: high-fidelity static design package and token references.
- `v2/`: previous V2 planning docs kept as implementation context, not durable root documentation.
- `steward/`: steward brief and Telegram polls used to resolve narrative choices.
- `workshop/`: workshop runbook and notes template.
- `prompts/`: prior design/synthesis/repo-refinement prompts.

These artifacts should archive with the hub when the implementation closes.
