# AI-Native Workflow Artifacts

This directory holds the reusable local artifacts for the AI-Native Developer Workflow hub.

- `agent-run-ledger.md` records delegated agent work.
- `workflow-scorecard.md` records measurable workflow outcomes.
- `agentic-house-rules.md` records the repo-local decisions agents should follow before adding new schemas, routes, UI, auth, or proof surfaces.
- `../reports/closeout-gate.md` records the agreement check before a lane is marked complete.

These files are templates plus the first evidence log. Keep execution truth in this hub and do not turn ad-hoc chat memory into repo truth.
